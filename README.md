#  Minha Estante

Sistema Web desenvolvido para o cadastro, organização e visualização de livros de uma biblioteca pessoal.


## Discentes: Emilha, Filipe, Luiz Augusto e Míriam.

##  Introdução

### Objetivo
Desenvolver uma aplicação web que permita ao usuário cadastrar seus livros, acompanhar o status de leitura e organizar seu acervo de forma prática e intuitiva.

### Relevância
O projeto busca facilitar a organização de livros pessoais, oferecendo uma interface simples e agradável para o gerenciamento da biblioteca.

### Justificativa
Além de resolver um problema cotidiano, o projeto permite aplicar conceitos fundamentais do desenvolvimento Front-End, como estruturação de páginas, estilização e interatividade.


## Desenvolvimento

### Ferramentas e Tecnologias

- HTML 
- CSS
- JavaScript
- Google Fonts (utilização de fontes para melhorar a identidade visual)
- Git (controle de versão durante o desenvolvimento)
- GitHub (hospedagem do codigo-fonte e colaboração no projeto)

### Funcionamento

O sistema possui três páginas principais:

- **Home:** apresentação do projeto.
- **Catálogo:** visualização dos livros cadastrados.
- **Administração:** cadastro de novos livros, contendo:
  - Título
  - Autor
  - Categoria
  - Status de leitura (Lido, Lendo ou Quero ler)
  - Descrição

Após o cadastro, os livros são exibidos no catálogo da aplicação.


##  Desafios Enfrentados

Durante o desenvolvimento, os principais desafios foram:

- Criar uma interface organizada e responsiva;
- Organizar a estrutura do projeto;
- Manipular os dados utilizando JavaScript;
- Manter a identidade visual consistente entre as páginas.


##  Trabalhos Futuros

Como o projeto atualmente possui foco no Front-End, algumas melhorias podem ser implementadas futuramente: 

- Back-End para armazenamento dos dados;
- Integração com um banco de dados;
- Sistema de login de usuários;
- Upload de capas dos livros;
- Pesquisa e filtros por categoria ou status;
- Integração com APIs de livros.
  

##  Funcionalidades

- Cadastro de livros;
- Organização do acervo;
- Visualização do catálogo;
- Controle do status de leitura;
- Interface responsiva.
